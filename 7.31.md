# java
* 不可以在方法内部定义静态变量
* 局部变量要被赋值才能使用
* 静态代码块在第一次新建只执行一
## 继承
* 继承实现类的扩展，关键字是extends
* 多态，上转型，父类比子类更抽象宽泛
* father s=new son();//多态，father是表现类型，son是实际类型
* 多态应用：依赖倒置原则，
* 在父类中构造器的方法，子承父类连构造器方法一起继承，继承父类 父类必须写无参的构造器，
* super()相当于调用父类的有参构造器
